# Diwali-Sales-Dashboard
Diwali Sales Dashboard-Power BI-Dashboard

Project Overview

This project analyzes Diwali sales data using Power BI to understand customer purchasing behavior, sales performance, and demographic trends. The dashboard provides interactive visualizations for better business decision-making during festive seasons.
________________________________________
Objectives

•	To analyze customer purchasing patterns during Diwali

•	To identify top-performing product categories

•	To understand sales distribution by age, gender, and occupation

•	To analyze region-wise and state-wise sales performance

•	To build an interactive and user-friendly dashboard
________________________________________
Tools & Technologies Used

•	Power BI – Dashboard creation

•	Microsoft Excel / CSV – Data handling

•	Power Query – Data cleaning and transformation
________________________________________

Dataset Description

The dataset contains sales transaction details during Diwali, including:

•	Customer ID

•	Gender & Age Group

•	Marital Status

•	Occupation

•	State & Region

•	Product Category

•	Orders & Sales Amount
________________________________________
Data Pipeline

1.	Collected dataset (Excel/CSV format)

2.	Imported data into Power BI

3.	Cleaned data using Power Query (handled missing values, formatting)

4.	Built interactive visuals and dashboard

5.	Finalized and published dashboard
________________________________________
Dashboard Features

•	KPI Cards:

o	Total Customers

o	Total Amount

o	 Total Orders

o	 Average Age

•	Visualizations:

o	Clustered Column Chart (Amount by State)

o	Clustered Column Chart (Amount by Gender and Marital_Status)

o	Donut Chart (Amount of age group,orders by category, Amount by Gender )

o	Treemap(Amount Occupation)

•	Filters (Slicers):

o	Central

o	Eastern

o	Northern

o	Southern

o	Western
________________________________________
Key Insights

•	Majority of buyers belong to the 26–35 age group

•	Female customers contribute more to total sales compared to males

•	Top occupations include IT, Healthcare, and Banking sectors

•	Certain states generate higher revenue, showing regional demand patterns

•	High sales observed in selected product categories during Diwali season
________________________________________

How to Run the Project

1.	Download dataset (Excel/CSV file)

2.	Open Power BI Desktop

3.	Load dataset into Power BI

4.	Perform data cleaning using Power Query

5.	Open the .pbix file

6.	Refresh data to view dashboard
